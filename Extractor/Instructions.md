### Installing Dependencies

## Requirements:
-Python
-xampp for mysql

## Instructions

1. initiate virtual evnironment (venv)

python -m venv venv

2. activate environment

venv\scripts\activate

3. install dependencies

pip install -r requirements.txt

4. locate the pdf yung iieextract

copy mo lang mismo path kasama yung file name and whatnot ex:
D:\programming study\Repositories\Extractor\ViewPdf.pdf

5. run extractor then follow prompts

python extractor.py

yung check_databases na scripts is for checking if yung extractor nakikita yung mismong mysql (if running) for  uploading
